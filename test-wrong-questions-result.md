# 错题练习结果显示测试

## 功能概述

错题练习提交后，结果页面只显示回答错误的问题，不显示已答对的问题。这样用户可以专注于错误的题目，提高学习效率。

## 实现方式

1. **ExerciseResult组件增强**：添加mode参数，支持'normal'和'wrong-questions'两种模式
2. **题目筛选**：在'wrong-questions'模式下，只显示回答错误的题目
3. **空结果处理**：如果所有题目都答对了，显示祝贺信息

### 1. 准备工作

2. 进入错题练习模式
3. 重新回答错题
4. 提交答案
